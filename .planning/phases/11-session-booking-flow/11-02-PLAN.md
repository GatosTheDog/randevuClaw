## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Gemini → executeTool | session_instance_id supplied by LLM; cross-tenant guard: business_id checked against context.business.id before any handler runs |
| client → book_session | clientPhone sourced from context (Telegram from.id), never from Gemini args — prevents impersonation |
