## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| booking request → enforcement pre-check | Client-supplied context (phone, business) must be authoritative; business.enforcementPolicy comes from the DB row (findBusinessByWebhookId), not from client input |
| cancel request → credit restore | Cancel path must verify the booking belongs to the requesting client before any status change or credit restore |
