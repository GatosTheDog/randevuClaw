       function hoursUntilSessionInAthens(
         sessionDate: string,   // "YYYY-MM-DD" Athens local
         sessionTime: string    // "HH:MM" Athens local 24h
       ): number

       Implementation — DST-safe approach using Intl:
       - Construct the Athens wall-clock instant as a UTC Date by using Intl.DateTimeFormat to measure the Athens offset at noon-UTC of sessionDate (same pattern used in athensEndOfDay in billing/queries.ts).
       - Noon UTC: const noonUTC = new Date(`${sessionDate}T12:00:00Z`);
       - Read Athens hour at noonUTC via Intl.DateTimeFormat('en-US', { timeZone: 'Europe/Athens', hour: '2-digit', hour12: false }).formatToParts(noonUTC). Extract the numeric hour.
       - offsetHours = athensHour - 12 (will be 2 for EEST winter, 3 for EEST summer).
       - Parse sessionTime HH:MM into hours and minutes.
       - sessionUTCMs = Date.parse(`${sessionDate}T${sessionTime.padStart(5,'0')}:00Z`) - offsetHours * 3_600_000
       - hoursUntil = (sessionUTCMs - Date.now()) / 3_600_000
       - return hoursUntil (may be negative if session is in the past — callers treat negative as "inside cutoff").

       No new library imports needed. No import of timezone.ts utilities (they do not expose this calculation).

    TypeScript should compile clean after this task alone.
