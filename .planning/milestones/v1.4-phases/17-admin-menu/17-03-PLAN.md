         `${session.sessionDate} ${session.sessionTime} — ${session.bookedCount}/${session.capacity} θέσεις`

         If sessions.length > 0:
           'Επερχόμενα μαθήματα (7 ημέρες):\n\n' + lines.join('\n')

           'Δεν υπάρχουν προγραμματισμένα μαθήματα για τις επόμενες 7 ημέρες.'

         [{ text: '🗑 Ακύρωση μαθήματος', callback_data: 'menu:classes:cancel_list' }]

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Telegram callback_data instanceId → cancelSession | Numeric instanceId from callback_data; ownership validated by cancelSession's FK subquery (sessionCatalog.businessId = business.id) |
| showCancelClassList → DB read | listSessions uses businessId from re-derived business object, not callback_data |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-17-10 | Tampering | Forged instanceId in menu:classes:cancel_yes callback | medium | mitigate | cancelSession validates ownership via FK chain: sessionInstances.catalogId → sessionCatalog.businessId = business.id. A foreign instanceId fails the WHERE clause and cancelSession returns false. |
| T-17-11 | Elevation of Privilege | Client sending menu:classes:cancel_yes callback | medium | mitigate | handleCallbackQuery menu dispatch guards with findBusinessByOwnerTelegramId; returns null for non-owners → no DB call |
| T-17-12 | Tampering | Replay of cancel_yes callback after cancellation | low | accept | cancelSession is idempotent — already-cancelled instances return false. Owner receives 'already cancelled' message. No data corruption. |
| T-17-13 | Information Disclosure | Session list exposes session count/capacity to crafted callback | low | accept | Session list is owner-only (admin menu path); owners are entitled to see all sessions for their business |
</threat_model>
